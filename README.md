# IRXBot — Industrial 4WD Robot

## Hardware
- 4× 57BL BLDC Motors with 50:1 gearbox
- 4× Flipsky 75100 VESC
- STM32 F446RE (CAN controller)
- NVIDIA Jetson Orin Nano (ROS2)
- 2× LiFePO4 12.8V 20Ah (25.6V total)

## Software Stack
- ROS2 Humble (Jetson)
- STM32CubeIDE (firmware)
- Nav2, SLAM Toolbox
